# snakewars26.github.io
SnakeWars est un jeu multijoueur en ligne de type Battle Royale où les joueurs contrôlent un serpent et s’affrontent dans une arène dynamique. Collectez des orbes pour grandir, utilisez des compétences stratégiques et éliminez vos adversaires pour être le dernier survivant. Jouable directement dans le navigateur, sans installation.
